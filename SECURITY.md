## 报告漏洞

如果你发现了安全漏洞，请**不要**通过公开 Issue 提交。

请通过以下方式私下报告：

1. 使用 GitHub 的 [私密漏洞报告](../../security/advisories/new) 功能
2. 或通过仓库维护者的联系方式私下联系

### 报告内容

请在报告中提供以下信息：

- 漏洞类型及描述
- 相关文件路径和代码位置
- 复现步骤
- 潜在影响评估

### 处理流程

1. 维护者将在 **7 个工作日** 内确认收到报告
2. 评估漏洞严重性并制定修复计划
3. 发布安全更新后，公开披露漏洞详情

感谢你帮助我们保持项目安全！
